# 🎨 LuxPalette Studio

> **Generador de 500 Paletas Procedurales & Rueda de Tríadas Interactiva**  
> *Creado por Ancient of Light | Una Producción de OmniumCore*

¡Bienvenido a **LuxPalette**, una herramienta web premium y de código abierto diseñada especialmente para diseñadores, programadores independientes y creadores de contenido! Esta aplicación te permite explorar el círculo cromático, calcular armonías en modo tríada y navegar de forma instantánea por una biblioteca masiva de 500 paletas de colores divididas por categorías.

---

## 🚀 Características Principales / Key Features

* **🎨 Rueda de Color Interactiva (Interactive Color Wheel):** Explora tonalidades de forma libre o activa el **Modo Tríada** para calcular tres colores armónicos automáticos en un triángulo perfecto de 120°.
* **⚡ 500 Paletas Procedurales (500 Dynamic Palettes):** Navegación ultra veloz optimizada a través de 14 categorías independientes (Pastel, Neón, Metálicos, Rojos, Azules, etc.).
* **🔍 Buscador en Tiempo Real (Real-Time Search):** Encuentra colores al instante buscando por nombre de tono, categoría o código Hexadecimal directo.
* **🌐 Selector de Idiomas (Language Toggle):** Cambia toda la interfaz entre Español e Inglés con un solo clic en milisegundos.
* **📋 Copiado Inteligente (Smart Clipboard):** Haz clic directamente sobre la línea de datos de cualquier tarjeta para copiar de forma aislada únicamente el valor que necesitas (`HEX`, `RGB` o `HSV`).
* **💳 Ventana de Soporte Integrada (Support Modal):** Interfaz limpia y no invasiva para recibir donaciones voluntarias globales y locales.

---

## 🛠️ Tecnologías Utilizadas / Tech Stack

Este proyecto fue desarrollado bajo una arquitectura frontend estática, ligera y optimizada para un alto rendimiento en navegadores:

* **HTML5 / JSX:** Estructura semántica de la aplicación y maquetación del modal interactivo.
* **CSS3 (Tailwind CSS):** Diseño responsivo en modo oscuro, efectos premium de difuminado de fondo (`backdrop-filter: blur`) y animaciones de expansión al pasar el cursor (`hover scaling`).
* **JavaScript (React / TypeScript):** Lógica matemática para el cálculo radial de la rueda en modo tríada, algoritmos para la generación procedural de las 500 paletas en tiempo real y gestión del portapapeles.
